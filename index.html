Forward — evaluate()

Takes a boolean expression string. Returns the full truth table, minterms, maxterms, satisfiability, tautology flag, and performance metrics.

Python
from core.evaluator import evaluate

table, metrics = evaluate("A.(B+C)")
print(table.minterms)      # [5, 6, 7]
print(table.satisfiable)   # True
print(metrics.eval_time_ms) # 0.21
←

Inverse — synthesize()

Takes a truth table. Returns the minimal boolean expression using Quine-McCluskey. Two equivalent expressions always reduce to the same minimal form.

Python
from core.synthesizer import synthesize

expr, metrics = synthesize(table)
print(expr)   # "A.C+A.B"

# A.B+A.!B always reduces to A
t, _ = evaluate("A.B+A.!B")
print(synthesize(t)[0])   # "A"

Deep Learning

Boolean algebra meets neural networks.

Every neuron is a soft threshold gate. This engine computes the crisp version — and that connection runs deeper than it looks.

Neural Networks as Boolean Circuits

A Binary Neural Network with ±1 weights is mathematically equivalent to a boolean circuit. evaluate() and synthesize() apply directly — find the exact logic function a BNN layer computes, not an approximation.

Formal Verification

For binary and quantized networks, proving a safety property holds for all inputs reduces to boolean satisfiability. The engine's SAT check is the primitive operation for neural network formal verification.

Interpretability

Given binary input features, find the minimal boolean expression describing when a model fires. Quine-McCluskey gives you the simplest human-readable rule — a concrete interpretability tool.

CUDA Acceleration Path

Truth table rows are fully independent — each maps 1:1 to a GPU thread. rows_evaluated is literally the thread count. GPU-accelerated boolean evaluation is the native compute primitive for large BNN layers.
